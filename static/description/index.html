SharePoint ODOO Connector

Document Management using SharePoint ODOO Connector

Overview

ODOO is a fully integrated suite of business modules that encompass the traditional ERP functionality. SharePoint is a document management and storage system. SharePoint is a web-based application that integrates with Microsoft Office and it is highly configurable and usage varies substantially between organizations. ODOO integration with SharePoint enhances operation of organization with legitimate documentation.

Key Features And Functions

  • Secure document management
  • Upload ODOO's document to SharePoint
  • Download SharePoint's document to ODOO
  • Directory structure management for multiple customers using ODOO
  • Manage seperate SharePoint account for each user
  • Support for both Enterprise and Community ODOO version
  • Support for multiple SharePoint sites
  • Support for custom documents library

SharePoint Credentials

  • SharePoint credentials management for each user.
  • SharePoint credentials verification using test Connection.
  • SharePoint subsite feature
    • User can upload/download files to his own created subsite
    • If no subsite is mentioned then default site will be used
  • SharePoint document library feature
    • User can upload/download files to their custom libraries
    • If no document library is mentioned then default "Documents" Library will be used
    • User having SharePoint account in a language other than English can use this feature. They just need to mention document library name in their language.

ODOO File Upload/Download

  • Integration of ODOO document with SharePoint for each customers
  • Multiple attachments
  • Download latest files from SharePoint to ODOO for each customer.
  • Download related files links from SharePoint to ODOO for each customer, and each customer can download file from the Download Document button when required.

ODOO Documents links

  • Download related files links from SharePoint to ODOO for each customer.
  • Each customer can download file from the Download Document button when required.

Documents on SharePoint

  • Directory structure for each customer at SharePoint.
  • Management of sites and document libraries on SharePoint.
